¿Qué es Git?

Git es un sistema de control de versiones. Permite guardar el historial de cambios de un proyecto, volver atrás si algo falla y trabajar de forma más segura.

Repositorio

Es la carpeta del proyecto que Git empieza a rastrear con git init.

Commit

Es una confirmación de cambios, como una foto del estado del proyecto en un momento determinado.

Branch

Es una rama de trabajo. Sirve para desarrollar cambios sin tocar la versión principal.

Comandos básicos

git init
git status
git add .
git commit -m "Primer commit"
git log

Estados de archivos

  • Untracked: Git todavía no rastrea el archivo.
  • Staged: el archivo fue agregado para el próximo commit.
  • Committed: el cambio ya quedó guardado en el historial.

Git ayuda a trabajar con orden, historial y seguridad.
