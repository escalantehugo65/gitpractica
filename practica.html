Actividad propuesta

Seguí estos pasos para publicar este sitio en GitHub desde tu equipo.

  1. Crear una carpeta del proyecto.
  2. Guardar dentro los archivos HTML y CSS.
  3. Abrir la terminal en esa carpeta.
  4. Inicializar Git.
  5. Hacer el primer commit.
  6. Crear un repositorio en GitHub.
  7. Conectar el remoto y hacer push.

Comandos para practicar

git init
git add .
git commit -m "Primer commit del sitio de clase"
git branch -M main
git remote add origin https://github.com/tu-usuario/tu-repo.git
git push -u origin main

Desafío para estudiantes

Agregar una nueva página llamada comandos.html con una lista de comandos de Git y enlazarla desde el menú de navegación.

La mejor forma de aprender Git es practicar con proyectos reales.
