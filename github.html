¿Qué es GitHub?

GitHub es una plataforma online donde podemos alojar repositorios Git, colaborar con otras personas y mantener un historial visible del proyecto.

Repos remotos

Podés guardar tu proyecto en la nube y acceder desde distintos equipos.

Issues

Sirven para registrar tareas, errores o mejoras pendientes.

Pull Requests

Permiten revisar y aprobar cambios antes de integrarlos a la rama principal.

Push directo vs Pull Request

Push directo Pull Request
Sube cambios sin revisión previa. Propone cambios para revisar antes del merge.
Más rápido. Más seguro en equipos.
Útil en proyectos personales o pequeños. Ideal para trabajo colaborativo.

Ejemplo de colaboración

Un desarrollador crea una rama, sube sus cambios y abre un Pull Request. Otra persona revisa el código y aprueba el merge a main.

GitHub agrega colaboración, revisión y publicación de proyectos.
